18 Words — iframe embed example

This page embeds the game exactly the way a third-party site would: a plain <iframe> pointing at one of the game's language folders. The game posts its content height to the parent window on load and on every resize, so the iframe below grows and shrinks automatically instead of showing scrollbars.

Embed code

<iframe
  src="https://YOUR_DOMAIN/en/index.html"
  title="18 Words game"
  loading="lazy"
  allow="clipboard-write"
  style="width:100%; max-width:480px; height:720px; border:0; border-radius:16px;"
></iframe>

Swap en/ for ru/ or th/ to embed a different language. allow="clipboard-write" lets the in-game "Copy Result" button work across origins.

Optional: auto-resize listener

window.addEventListener("message", (event) => {
  if (event.data && event.data.type === "18words:resize") {
    document.getElementById("game").style.height = event.data.height + "px";
  }
});

The game only posts this message when it detects it is running inside an iframe (window.self !== window.top), so it stays a no-op when played standalone.
